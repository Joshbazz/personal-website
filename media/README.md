# Media Directory

This directory contains all images and media files for the portfolio website.

## Directory Structure

```
media/
├── profile/
│   └── PROFILE_PICTURE.png      # Your profile photo (recommended: 400x400px)
│
└── projects/
    ├── polymarket.png            # Polymarket Trade Collector screenshot
    ├── lstm-trading.png          # LSTM Trading Model visualization
    ├── neural-network.png        # Neural Network Trading System
    ├── nfl-analysis.png          # NFL Market Analysis charts
    ├── volatility-models.png     # Implied Volatility Models
    ├── portfolio-optimizer.png   # Portfolio Optimization Tool
    ├── lookahead-bias.png        # Lookahead Bias Research
    ├── crypto-lending.png        # Crypto Lending Analytics
    ├── data-pipeline.png         # Market Data Pipeline
    └── pricing-engine.png        # Relative Value Pricing Engine
```

## Image Specifications

### Profile Picture
- **Filename:** `PROFILE_PICTURE.png` (or .jpg)
- **Size:** 400x400px recommended (square)
- **Location:** `media/profile/`

### Project Images
- **Size:** 800x500px recommended (16:10 aspect ratio)
- **Format:** PNG or JPG
- **Location:** `media/projects/`

## How to Add Images

1. Drop your image file into the appropriate folder
2. Rename it to match the expected filename (see structure above)
3. The website will automatically display the image
